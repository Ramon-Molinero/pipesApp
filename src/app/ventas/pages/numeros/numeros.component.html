Pipes incluidos en Angular - Usualmente en Common Module

Formatea un valor de acuerdo con las opciones de dígitos y las reglas de configuración regional.


ventasNetas | number

{{ventasNetas}} = {{ventasNetas | number}}


ventasNetas | number : ' 1.2-2 '

{{ventasNetas}} = {{ventasNetas | number:'1.2-2'}}


ventasNetas | number : ' 1.2-2 ' : ' fr '

{{ventasNetas}} = {{ventasNetas | number:'1.2-2':'fr'}}
Transforma un número en una string de moneda, formateada de acuerdo con la configuración regional.


ventasNetas | currency

{{ventasNetas}} = {{ventasNetas | currency}}


ventasNetas | currency : ' CAD '

{{ventasNetas}} = {{ventasNetas | currency:'CAD'}}


v.n | currency : ' EUR ' : 'symbol-narrow' : '1.4-4'

{{ventasNetas}} = {{ventasNetas | currency:'EUR':'symbol-narrow':'1.4-4'}}

porcentaje | currency

porcentaje | currency : ' 1.2-2 '

{{porcentaje}} = {{porcentaje | percent: '1.2-2'}}


porcentaje | currency : ' 1.3-3 '
